# Pratap Adhikari - VS Code Themed Portfolio

A unique, interactive developer portfolio designed to replicate the visual experience of Visual Studio Code. Built with modern web technologies, it showcases my skills, experience, and projects in a familiar developer environment.

- **VS Code Aesthetic**: Authentic look and feel with Activity Bar, Sidebar, Tabs, Breadcrumbs, and Status Bar.
- **Theme System**: 
  - **Dark+ (Default)**: The classic VS Code dark experience.
  - **Light Modern**: A clean light theme.
  - **High Contrast**: Enhanced accessibility with a pure black/white theme.
- **Responsive Design**: 
  - Collapsible Sidebar ("Explorer") that adapts to mobile screens.
  - Smooth layouts that work on all devices.
- **Interactive Navigation**:
  - File Explorer navigation to About, Experience, Projects, and Contact pages.
  - "Tab" system showing active pages.
- **Accessibility**:
  - Font size toggles (A, A+, A++).
  - High contrast mode.
